WPF + PowerShell GUI

Componentes drop-in para inyectar GUI WPF + mini consola en cualquier script .ps1 existente. Sin compilacion, sin dependencias.

00TL;DR

flow.ps1 -STA
  Add-Type WPF
  [xml]$xaml = @'...'@
  XamlReader.Load -> $window
  FindName -> $ctrl hashtable
  $btn.Add_Click({ ... })
  $window.ShowDialog()
keyWPF requiere STA. PS5.1 ya lo trae. PS7 necesita -STA explicito.

08Runspaces async

powershellfunction Start-AsyncJob {
    param([scriptblock]$Work, [hashtable]$Args = @{})
    $rs = [runspacefactory]::CreateRunspace()
    $rs.ApartmentState = 'STA'
    $rs.Open()
    $rs.SessionStateProxy.SetVariable('sync', $sync)
    foreach ($k in $Args.Keys) { $rs.SessionStateProxy.SetVariable($k, $Args[$k]) }
    $ps = [powershell]::Create()
    $ps.Runspace = $rs
    [void]$ps.AddScript($Work)
    [void]$ps.BeginInvoke()
}

11Ejecutor + key handler

powershell$script:CmdHistory   = New-Object System.Collections.Generic.List[string]
$script:CmdHistoryIdx = 0

function Invoke-ConsoleCommand {
    param([string]$Cmd)
    $rs = [runspacefactory]::CreateRunspace()
    $rs.ApartmentState = 'STA'; $rs.Open()
    $rs.SessionStateProxy.SetVariable('sync', $sync)
    $rs.SessionStateProxy.SetVariable('cmd',  $Cmd)
    $ps = [powershell]::Create()
    $ps.Runspace = $rs
    [void]$ps.AddScript({
        try {
            $r = Invoke-Expression $cmd 2>&1
            if ($null -ne $r) {
                $t = ($r | Out-String).TrimEnd() + "`r`n"
                $sync.Window.Dispatcher.Invoke([action]{
                    $sync.ConsoleOut.AppendText($t); $sync.ConsoleOut.ScrollToEnd()
                })
            }
        } catch {
            $sync.Window.Dispatcher.Invoke([action]{
                $sync.ConsoleOut.AppendText("ERR: " + $_.Exception.Message + "`r`n")
            })
        }
    })
    [void]$ps.BeginInvoke()
}

$ctrl.ConsoleIn.Add_KeyDown({
    param($s, $e)
    switch ($e.Key) {
        'Return' {
            $cmd = $s.Text.Trim()
            if (-not $cmd) { return }
            $script:CmdHistory.Add($cmd) | Out-Null
            $script:CmdHistoryIdx = $script:CmdHistory.Count
            $ctrl.ConsoleOut.AppendText("PS> $cmd`r`n")
            $ctrl.ConsoleOut.ScrollToEnd()
            $s.Clear()
            Invoke-ConsoleCommand $cmd
            $e.Handled = $true
        }
        'Up' {
            if ($script:CmdHistoryIdx -gt 0) {
                $script:CmdHistoryIdx--
                $s.Text = $script:CmdHistory[$script:CmdHistoryIdx]
                $s.CaretIndex = $s.Text.Length
                $e.Handled = $true
            }
        }
        'Down' {
            if ($script:CmdHistoryIdx -lt ($script:CmdHistory.Count - 1)) {
                $script:CmdHistoryIdx++
                $s.Text = $script:CmdHistory[$script:CmdHistoryIdx]
                $s.CaretIndex = $s.Text.Length
                $e.Handled = $true
            } else { $s.Clear() }
        }
    }
})

$ctrl.ConsoleClear.Add_Click({ $ctrl.ConsoleOut.Clear() })
$window.Add_Loaded({ $ctrl.ConsoleIn.Focus() })
seguridadInvoke-Expression ejecuta cualquier cosa. OK para input local del propio user. NUNCA usar con input remoto sin sanitizar.

14CSV drop-in (integracion)

Cada fila = pieza autonoma para inyectar en un script .ps1 existente. Columnas:

ColSignificado
idIdentificador unico
kindxaml | powershell | cmd
insert_atDonde insertarlo en el script existente (ancla)

snippetCodigo listo (saltos como \n, " escapado como "")

Anclas reconocidas:

  • xaml.window-resources: dentro de <Window.Resources>
  • xaml.grid-row: una fila adicional al Grid root + entrada en RowDefinitions
  • xaml.action-bar: dentro del StackPanel de botones de accion
  • ps.before-show: antes de $window.ShowDialog()
  • ps.controls-bind: dentro del bucle foreach ... FindName
  • ps.sync-fields: dentro de [hashtable]::Synchronized(@{ ... })
  • ps.helpers: junto al resto de funciones helper
  • ps.event-wire: tras carga de XAML, antes de ShowDialog
